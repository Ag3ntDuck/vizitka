Cистема бронирования подарков для друзей и семьи

Firebase Auth (email/Google), Firestore, TypeScript Реализовала полноценную CRUD-систему: добавление, удаление, бронирование подарков. Есть условная соцсеть, можно регистрироваться, менять пароль, добавлять подарки по разным категориям, в зависимости от желаемости. Так же пользователь не видит подарки, которые забронировали у него, зато может видеть если у кого-то другого забронировали, чтобы не было ситуаций двух одинаковых подарков.

Chrome-расширение (Chrome Extensions API) для напоминания полива растений. Напоминалки работают через chrome.alarms (внутренний счётчик). Данные хранятся в localStorage. Уведомления показываются кастомным окошком внизу экрана. Можно добавлять растения, указывать им как часто нужно поливать, в зависиомти от даты нужного полива, полив будет гореть разными цветами.
